## [1. 本实验的目标](https://codelabs.developers.google.com/codelabs/getting-started-with-spec-driven-development-in-antigravity?hl=zh-cn#0)

在本实操实验中，您将学习如何使用 [Google Antigravity](https://antigravity.google/docs/get-started?hl=zh-cn) 创建应用，并将其部署到 Google Cloud 中。本实验还将向您介绍规范驱动开发的概念。

## 学习内容

- 了解 [Google Antigravity](https://antigravity.google/docs/get-started?hl=zh-cn) 的基础知识。
- 了解规范驱动开发的基础知识
- 了解如何在 [Cloud Run](https://cloud.google.com/run?hl=zh-cn) 中轻松部署应用。
